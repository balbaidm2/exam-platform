# منصة اختبار القدرات التشخيصي

منصة إلكترونية لتشخيص مستوى الطلاب في اختبار القدرات العامة.

## المميزات

- ✅ 80 سؤالاً (40 لفظي + 40 كمي)
- ⏱️ مؤقت 120 دقيقة
- 📊 لوحة معلم لمتابعة النتائج
- 💾 حفظ تلقائي للنتائج في قاعدة البيانات
- 📥 تصدير النتائج إلى Excel
- 🎨 تصميم احترافي عربي

## المتغيرات البيئية المطلوبة

عند النشر على Vercel، أضف هذه المتغيرات:

- `VITE_SUPABASE_URL` - رابط مشروع Supabase
- `VITE_SUPABASE_ANON_KEY` - مفتاح anon public
- `VITE_TEACHER_PASSWORD` - كلمة مرور لوحة المعلم

## التقنيات

- React 18
- Vite
- Tailwind CSS
- Supabase
- React Router

- الصفحة الرئيسية: `/`
